# Typescript JWT Auth

1. Install "devDepedencies" and "depedencies"
```
npm install
```
will only install "dependencies"
```
npm install --production
```
will only install "devDependencies"
```
npm install --dev
```
2. Start app in production
```
npm run build
npm run start
```
Start app in the development
```
npm run build
npm run dev
```
